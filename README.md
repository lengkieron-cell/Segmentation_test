This project was my first attempt at creating a deep learning pipeline for semantic segmentation. This code , which was originally made for roof segmentation , served as a proof of concept for other high accuracy feature extraction. The framework serves as a base architecture for my current transition to Medical Imaging , where I hope to transfer principles from this “practice” project to address class imbalances and identification inaccuracy in the clinical field.
